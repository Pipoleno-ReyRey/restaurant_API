# restaurant_API
This project is a API to the dishes of a restaurant builded in C#-.NET 8 and MySQL.

# Instructions
Clone the repository.
API.

# EndPoints

Food

GET /api/getFood
GET /api/getDish/{id}
POST /api/postDish
PUT /api/editDish/{id}
DELETE /api/deleteDish/{id}

Orders

GET /api/getOrders
POST /api/postOrder/{nameCustomer}+{dishes}
